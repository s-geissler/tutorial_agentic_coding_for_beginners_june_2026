# Overview

This is a repository containing the course materials for a 90 minute lecture on the basics of agentic coding and the AI driven generation of code, HTML-based presentation slides, and scientic writing.

The content is split into the course slides generated using deck.js (https://github.com/s-geissler/deck.js) in `./html_slides`. A local deck.js checkout is included in the `./materials` folder. Where example code and projects generated in the context of this course lives.
