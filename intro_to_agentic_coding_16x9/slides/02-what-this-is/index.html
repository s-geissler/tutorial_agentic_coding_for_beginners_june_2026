What this is, and what it isn't

✓ This is

A snapshot of how agentic coding tools work as of June 2026: a brief background and a hands-on tour of opencode + Matt Pocock's skill collection. The goal is to provide enough information so you can decide for yourself what's worth your time.

✗ This is not

A vibe-coding tutorial.
A "use ChatGPT for everything" pitch.
A survey of every agentic tool on the market.
A model-finetuning lecture.

Agenda

  • LLM basics: tokens, context windows, and why models only see text
  • Harness anatomy: prompts, tool schemas, files, history, and feedback loops
  • Agentic concepts: skills, agents, subagents, and when each helps
  • Tool landscape: why opencode, and how its terminal workflow feels
  • Setup path: install opencode, add skills, and inspect the TUI
  • Workshop loop: grill, TDD, PRDs, issues, and Ralph-style iteration
