What you'll build on

A small, deliberately imperfect FastAPI todo app. JSON file storage, vanilla-JS frontend, no tests for the API layer. About 100 lines.

backend

FastAPI · ~80 lines

Five endpoints: list, create, read, update, delete. Pydantic models. JSON file persistence in data/todos.json.

frontend

Vanilla JS · ~60 lines

Add a todo, mark it done. No delete button. No tests. Plain CSS. Designed to be improvable in 15 minutes.

tests

pytest · 5 tests, storage layer only

Storage layer is covered. The API layer is not. The hands-on will fix that with /tdd.
