Building slides with an agent is meta — deck.js built this very deck. The same four-step progression, with one extra: learn the design language first. Click an exercise for the detail.

Common pitfalls
  • Skip DESIGN.md — off-brand decks, every agent invents a new look.
  • Skip the grill — doc-dump decks with no claims, no narrative.
  • Skip the slice — one mega-HTML file, hand-edited, breaks the build.

This deck — the one you’re looking at — was built with the discipline on this slide. The first revision was one HTML file. The current revision is 26 folders, each a vertical slice.
