Count the hits, multiply by 4, and you have π

The whole method is a short loop: throw a dart, test whether it lands inside the quarter circle, and tally the hits.

import random

N     = 100_000          # number of darts to throw
hits  = 0

for _ in range(N):
    x = random.random()   # uniform in [0, 1)
    y = random.random()
    if x*x + y*y <= 1:    # inside the quarter circle?
        hits += 1

pi_hat = 4 * hits / N      # estimate of pi
1

Sample

Draw x and y independently and uniformly from [0, 1) — a random point in the unit square.

2

Test

A point is a hit when x² + y² ≤ 1 — the squared distance from the origin is at most 1.

3

Estimate

The hit ratio hits / N approximates π / 4, so multiply by 4. Error scales as 1/√N.
