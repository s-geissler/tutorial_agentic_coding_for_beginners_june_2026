Keyboard, TOC, dark mode, and progress — built in

Navigation never gets in the way. Every feature is optional and unobtrusive.

→ ← Advance / retreat

Arrow keys, Page Up/Down, and Space advance or retreat. The buttons in the top-right bar work too.

Esc Table of contents

Opens an overlay listing all slides by their data-toc attribute. Click any entry to jump. Also works via the top-right button.

T Theme toggle

Switches between light and dark mode. Preference is persisted to localStorage and respects prefers-color-scheme on first visit.

Home / End

Jump to the first or last slide. The progress bar at the top shows your position. The section label keeps context visible.

Slide counter

Top-right corner shows 03 / 14 format. Hover to see it more clearly.

Zero-config setup

All navigation is driven by data-* attributes on each slide. The nav.js script fills headers and builds the TOC automatically.
