Build extracts and bundles — no transpiler needed

A single command produces a portable dist/index.html with all assets copied in.

dist/index.html
Self-contained
Opens via file://

Tree copy, not bundling

The build copies assets/ and brand/ directories verbatim. No CSS/JS inlining — native <link> and <script> tags.

Path rewriting

Slide-local paths like assets/hero.png are rewritten to dist/assets/slides/03-name/hero.png automatically.
