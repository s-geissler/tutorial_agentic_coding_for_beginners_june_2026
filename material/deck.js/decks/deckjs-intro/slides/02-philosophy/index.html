Slides are authored in HTML, not locked in a black box

deck.js treats presentations as what they are: structured documents with a visual layer.

No dependencies

A single deck.js file at the repo root. Zero npm packages, no node_modules, no build step required to preview slides.

Works anywhere

Built decks open via file:// — no HTTP server needed. Take them offline, email them, put them on a USB drive.

Open format

Every slide is standalone HTML. You can open, edit, diff, and version-control slides like any other code. No proprietary editor.

Local-first

Author on your machine, build locally, no cloud dependency. Assets are bundled into the output — no CDN required.
