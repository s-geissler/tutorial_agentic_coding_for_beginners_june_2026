Three commands to go from zero to a working deck

deck.js has no package manager. Clone the repo and run:

1. Create a new deck

node deck.js new my-talk

Vendors the default template, theme, and scaffold files.

2. Add slides

node deck.js add 01-title --scaffold=title
node deck.js add 02-overview --scaffold=default
node deck.js add 03-architecture --scaffold=picture

Each scaffold creates slides/NN-name/index.html plus style.css and script.js.

3. Build and open

node deck.js build

Produces dist/index.html. Open it in any browser, or push it to GitHub Pages / a static host.

✓

That's it

No npm install. No build server. No lockfile. Your slides are just HTML — and they'll still render in ten years.
