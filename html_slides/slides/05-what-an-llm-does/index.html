What an LLM actually does

One job. Repeated billions of times.

iteration 1 — waiting

A deep neural network of transformer blocks. Given all the text so far, the model assigns a probability to every possible next token and samples one (the highlighted row). The predicted token is appended to the context, and the loop continues — one token at a time.

That's it. No retrieval, no reasoning engine, no planning module, no memory beyond the window. Everything else you see in an "agent" is added around the model.
